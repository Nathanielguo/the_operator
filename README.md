## The Operator

营销 × 产品 × 战略 AI 工作台

将模糊的问题，转化为清晰的战略与可执行方案。

⸻

项目简介

The Operator 是一款面向产品经理、市场营销人员、创业者、咨询顾问以及 AI 深度用户打造的 AI 工作台。

它将经典商业框架与大语言模型结合，通过结构化工作流帮助用户快速完成：

* 战略分析
* 竞品研究
* 品牌搭建
* 文案创作
* 营销策划
* 产品规划
* PRD 编写
* Roadmap 排序
* 多视角洞察分析

无需安装。

打开网页即可使用。

⸻

核心功能

01｜Strategy Canvas

通过经典商业框架快速分析问题：

* STP 分析
* SWOT 分析
* 4P 营销组合
* Porter 五力模型
* BCG 矩阵
* Ansoff 增长矩阵

⸻

02｜Competitive Intel

竞品情报分析中心：

* 竞品对标
* SWOT 分析
* 定位地图
* 差异化机会识别
* 白空间发现

⸻

03｜Brand Architect

品牌体系构建工具：

* 品牌定位
* 品牌金字塔
* 品牌人格
* Tone of Voice
* 价值主张

⸻

04｜Copy Studio

AI 文案工作室：

支持生成：

* 品牌 Slogan
* 产品标语
* 落地页标题
* 小红书文案
* 社交媒体内容
* 广告创意

⸻

05｜Campaign Planner

营销活动策划工具：

输出完整营销方案：

* 目标设定
* 用户分析
* 核心信息
* 渠道规划
* KPI 设计
* 预算建议

⸻

06｜Product Discovery

产品发现模块：

* JTBD 分析
* 用户痛点提炼
* 机会点挖掘
* 用户需求洞察

⸻

07｜PRD Drafter

PRD 自动生成工具：

支持生成：

* 项目背景
* 产品目标
* 用户故事
* 功能需求
* 验收标准
* 风险评估

⸻

08｜Roadmap & Prioritize

需求优先级排序工具：

内置：

* RICE
* ICE
* Kano
* MoSCoW
* WSJF

等主流产品框架。

⸻

09｜Insight Lab

多视角分析实验室：

同一个问题同时从：

* Strategist（战略视角）
* Marketer（营销视角）
* Product Manager（产品视角）

进行分析。

帮助发现认知盲区。

⸻

支持模型

支持自定义 API Key。

目前支持：

* Claude
* GPT
* Gemini
* DeepSeek
* Qwen
* Kimi
* GLM
* Doubao
* MiniMax

所有 API Key 仅保存在浏览器本地。

不会上传至服务器。

⸻

使用方式

本地运行

1. 下载项目文件
2. 打开 index.html
3. 配置 API Key
4. 选择模型
5. 开始使用

⸻

GitHub Pages

项目支持直接部署至 GitHub Pages。

无需服务器即可在线访问。
